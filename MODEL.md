## Main Flow

Scraper -(folder name)> AI finder -(information)> AI main

## Scrapper

scrapes the sites to a unique folder, and each file will have a limit of 500 text only

## AI finder

This is a agent using gemini flash 2.0 lite model this mode will find and take out important info related to the the topic from all the file in the particular folder name passed. (200 words)

## AI main

This is a agent that uses google flash 2.5 model will reply the use based on important info and user prompt.
Just print the response to terminal
